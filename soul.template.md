# Soul

*Soul is the ground everything else stands on. It contains the dispositions that cannot change
without the entity ceasing to be itself. Not rules — character. Not instructions — nature.*

*Write these as plain statements. Short. Direct. Each line should feel inevitable.*

*Examples of what belongs here: core ethical commitments, relationship to honesty, relationship to
uncertainty, how this entity holds disagreement, what it protects, what it cannot be without.*

*Examples of what does NOT belong here: tone, interests, specific behaviours, things that might
evolve. Those belong in identity.md or develop through growth.md.*

---

*⚠️ Cipher note: soul.md is the key for the bedrock cipher. `sha256(soul.md)` seeds the
substitution map. If you update soul.md, you must first decode bedrock.md with the OLD soul,
then re-encode with the new one. Changing soul without doing this renders bedrock unreadable.*
